## 🚀 როგორ დავიწყოთ მუშაობა?

### 1. დოკუმენტაციის ნახვა Obsidian-ში
გახსენით ეს საქაღალდე თქვენს Obsidian-ში (როგორც ახალი ვოლტი - Vault), რათა ვიზუალურად დაინახოთ დეპარტამენტები და სქემები.

### 2. საჭირო ბიბლიოთეკების დაყენება
აგენტური მორბენალის გამოსაყენებლად ტერმინალში უნდა დააყენოთ შემდეგი ბიბლიოთეკები:
```bash
pip install google-generativeai anthropic python-dotenv
```
*(რეკომენდებულია ამის გაკეთება ვირტუალურ გარემოში - venv, რათა თავიდან აიცილოთ ვერსიების კონფლიქტი).*

### 3. API გასაღებების კონფიგურაცია
შექმენით ფაილი `.env` პროექტის ძირში (ან `Central AI Team` საქაღალდეში) და ჩაწერეთ თქვენი გასაღები:
```env
GEMINI_API_KEY=თქვენი_gemini_გასაღები
# ან
ANTHROPIC_API_KEY=თქვენი_claude_გასაღები
```

### 4. დისპეტჩერის გაშვება (Idea Routing)
გახსენით ტერმინალი, გადადით `Central AI Team` საქაღალდეში და გაუშვით დისპეტჩერი თქვენი ახალი იდეის დასარუტებლად:
```bash
cd "Central AI Team"
python dispatcher.py "აღწერეთ თქვენი იდეა აქ"
```
ეს ბრძანება შექმნის აქტიურ სადისპეტჩერო ბარათს `Idea Dispatcher Output.md` Obsidian-ში.

### 5. აგენტების გაშვება (Agent Runner)
სადისპეტჩერო ბარათის Checklist-ის ეტაპობრივად შესასრულებლად და კოდის გენერირებისთვის გაუშვით მორბენალი:
```bash
cd "Central AI Team"
python agent_runner.py
```
ყოველი ნაბიჯის შემდეგ სკრიპტი მოგთხოვთ დასტურს (`Y/N`), სანამ კოდს რეალურად ჩაწერს ფაილში.
