## 🔌 როგორ იყენებენ აგენტები

> Agent Reach „გლუვი ფენაა" — თვითონ არ კითხულობს, აყენებს/აკონფიგურირებს upstream ხელსაწყოებს და skill-ით ასწავლის აგენტს მათ პირდაპირ გამოყენებას.

1. **Claude Code** — ავტომატურად: skill დარეგისტრირებულია `~/.claude/skills/agent-reach`-ში. URL-ის ან „მოძებნე ინტერნეტში X" მოთხოვნისას თავად ააქტიურდება.
2. **პირდაპირი პატერნები (ნებისმიერი აგენტი/სკრიპტი):**
   ```
   curl -s "https://r.jina.ai/<URL>"        # ვებგვერდი → სუფთა markdown (ტესტირებულია ✅)
   yt-dlp --skip-download --write-auto-subs <YT_URL>   # YouTube სუბტიტრები
   agent-reach doctor                        # არხების დიაგნოსტიკა
   agent-reach configure                     # გასაღებების დამატება
   ```
3. **MCP server** — `agent_reach/integrations/mcp_server.py` (Antigravity-სთვის / Telegram ბოტისთვის მომავალში).
