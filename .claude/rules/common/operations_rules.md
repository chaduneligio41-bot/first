# 📋 Agentic Operations SOP Guide

ეს ფაილი განსაზღვრავს საოპერაციო წესებს (Standard Operating Procedures) პროექტის ფარგლებში AI აგენტებისა და ქვე-აგენტების ეფექტურად სამართავად.

---

## 🛠️ 7 აგენტური ეტაპი (7 Agentic Pillars)

### 1. ⚡ სწრაფი რეაგირება (`Just Ask`)
* მცირე შეკითხვებისას ან მარტივი კოდის კორექტირებისას იმოქმედე სწრაფად, დამატებითი გეგმების გარეშე.

### 2. 💾 განმეორებადი ცოდნის ავტომატიზაცია (`Skill`)
* ნებისმიერი განმეორებადი ამოცანა (ტესტების გაშვება, მონაცემთა ბაზის შემოწმება) უნდა გარდაიქმნას Skills სკრიპტებად.

### 3. 🤖 რთული კვლევების დელეგირება (`Subagent`)
* რთული ან მოცულობითი ამოცანების დროს (მაგ. დეტალური დიზაინ-აუდიტი ან უსაფრთხოების რევიუ) შექმენი და გამოიძახე სპეციალიზებული სუბაგენტები, რათა არ გადაიტვირთოს ძირითადი ჩატის კონტექსტი.

### 4. 👥 სპეციალიზებული აგენტების გუნდი (`Agent Team`)
* მრავალმხრივი ამოცანებისას (UI/UX + API + Database) მართე სუბაგენტების კოორდინირებული გუნდი.

### 5. 🎯 სრული ავტონომია და გადამოწმება (`/goal`)
* დიდი მოცულობის სამუშაოების შესრულებისას იმოქმედე ავტონომიურად, გადაჭერი ერორები და დაატესტე წარმატებულ დასრულებამდე.

### 6. 📈 სისტემური მართვა (`Workflow`)
* კომპლექსური ამოცანების დაწყებისას შექმენი `task.md` (Checklist) და ყოველი ეტაპის დასრულებისას მონიშნე შესრულებული პუნქტები `[x]`.

### 7. 🔗 გარე სამყაროსთან კავშირი (`MCP / CLI`)
* გამოიყენე ლოკალური CLI ხელსაწყოები და MCP სერვერები ფაილებთან, სკრიპტებთან და ბრაუზერთან უსაფრთხო ინტერაქციისთვის.
